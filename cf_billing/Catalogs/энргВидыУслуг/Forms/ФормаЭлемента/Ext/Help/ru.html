Справочник "Виды услуг"

Справочник "Виды услуг" предназначени для хранения информации по услугам, предоставляемым организацией.

Справочник является иерархическим, поэтому виды услуг могут объединяться в группы. Для создания группы необходимо нажать кнопку «Создать группу» в форме списка справочника. Затем в форме создания группы ввести ее наименование и родителя.

Для создания нового вида услуг необходимо нажать кнопку «Создать» в форме списка соответствующего справочника. Далее заполняем реквизиты справочника:

"Группа" - группа, к которой относится данная услуга, "папка", в которой она находится при открытии справочника. Если поле "Группа" пустое, значит, элемент располагается в корне справочника).

"Наименование" - наименование услуги. Если услуга составная, рекомендуется называть ее так, чтобы было понятно из каких услуг она состоит, например: "ГВС + отопление"

"Код" - внутренний идентификатор системы, присваивается системой автоматически. Изменять его вручную не нужно

"Сезонность" - наличие у услуги привязанности к какому-либо периоду. Отопительному, для отопление и поливному - для полива

"Завист от услуги" - заполняется коммунальной услугой, если одна считается на основании другой, например для ГВС двухкомпонентной, Теплоэнергия, затрачиваемая на подогрев, зависит от объемов потребления Теплоносителя

"Тип услуги" - выбираем из списка необходимый тип услуги: коммунальные, прочие, разовые

"Товар" - в реквизите необходимо указать товар, которым измеряется услуа

"Вид площади" - указывается для услуг, нормивное потребление которых рассчитывается исходя из площади. Например, отопление.

"Ед. изм" - единица измерения, которая мериет услугу

"Вид площади МКД" - заполняется в случае, если начисление на ОДН при отсутствии ОДПУ зависит от площади мест общео пользования. В выпадающем списке для таких услуг необходимо выбрать "Площадь мест общего пользования"

"Использовать повышающий коэффициент" - признак устанавливается в случае использования при начислении на данную услугу повышающего коэффициента

"Индивидульные начисления зависят от наличия общедомового прибора учета" - признак устанавливается в случае зависимости индивидуальных начислений от наличия или отсутствия ОДПУ по данной услуге. Например, отопление.

"Норматив только для индивидуальных" - признак устанавливается на услугу при использовании одного и того же значения норматива для индивидуальных и общедомовых услуг. Например, Теплоэнергия на подогрев

"Уменьшать базу распределения по общедомовому прибору учета только есть все абоненты оборудованы индивидуальным прибором учета" - признак устанавлиается в случае уменьшения распределяемого объема на величину индивидуальных потреблений, только при наличии ИПУ у всех абонентов МКД. Например, отопление.

"Составная услуга" - признак устанавливается в случае, если услуга является составной и включает в себя несколько простых услуг. При установке признака, становится доступной табличная часть в нижней части формы, в которой необходимо указать простые услуги, входящие в составную.
