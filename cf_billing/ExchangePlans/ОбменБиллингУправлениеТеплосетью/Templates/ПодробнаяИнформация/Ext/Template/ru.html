Синхронизация данных с 1С:Управление теплосетью, редакция 2.3 предназначена для настройки и выполнения синхронизации данных между конфигурациями "1С:Биллинг, редакция 1.1" по правилам обмена. Использование правил обмена позволяет производить гибкую настройку и реализовывать  сложные сценарии синхронизации данных. Создавать и редактировать правила обмена можно в конфигурации 1С: Конвертация данных, редакция 2.0. В этой синхронизации данных наглядно демонстрируются основные возможности подсистемы Обмен данными, такие как:

  • Использование различных вариантов подключения к другой программе.
  • Сопоставление объектов информационных баз, позволяющее указать, какие объекты являются одинаковыми для двух синхронизирующихся программ.
  • Синхронизация данных с использованием правил конвертации.
  • Настройка правил синхронизации данных, ограничивающих миграцию данных между двумя программами.
  • Автоматическая синхронизация данных по заранее заданному расписанию.
